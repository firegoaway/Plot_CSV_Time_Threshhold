# 🔄 Таблица преобразования единиц измерения для пожарной безопасности

## 1. Оптическая плотность и коэффициент экстинкции

### Основные соотношения

| Величина | Обозначение | Ед. изм. | Формула |
|----------|-------------|----------|---------|
| Коэффициент экстинкции | K | 1/м | Базовая величина |
| Оптическая плотность (Нп/м) | μ_Np | Нп/м | μ_Np = K |
| Оптическая плотность (дБ/м) | μ_dB | дБ/м | μ_dB = 10·log₁₀(e^K) = 4.343·K |
| Оптическая плотность (FDS) | μ_FDS | 1/м | μ_FDS = K/2.3 = K·log₁₀(e) |
| Затенение (FDS) | OD_% | %/м | OD_% = (1 - e^{-K})·100 |
| Затенение (десятичное) | OD | — | OD = 1 - 10^{-μ_FDS} |

### Формулы преобразования

#### Из K [1/м]

| В какую величину | Формула | Пример: K=0.05 |
|------------------|---------|----------------|
| Нп/м | μ_Np = K | 0.05 Нп/м |
| дБ/м | μ_dB = 4.343·K | 0.217 дБ/м |
| FDS OD [1/м] | μ_FDS = K/2.3 | 0.0217 1/м |
| FDS %/м | OD_% = (1-e^{-K})·100 | 4.88 %/м |
| % затенения | OD = (1-10^{-μ_FDS})·100 | 4.88% |

#### Из μ_Np [Нп/м]

| В какую величину | Формула | Пример: μ_Np=0.023 |
|------------------|---------|-------------------|
| 1/м (K) | K = μ_Np | 0.023 1/м |
| дБ/м | μ_dB = 4.343·μ_Np | 0.10 дБ/м |
| FDS OD [1/м] | μ_FDS = μ_Np/2.3 | 0.01 1/м |
| FDS %/м | OD_% = (1-e^{-μ_Np})·100 | 2.27 %/м |

#### Из μ_dB [дБ/м]

| В какую величину | Формула | Пример: μ_dB=0.2 |
|------------------|---------|------------------|
| 1/м (K) | K = μ_dB/4.343 | 0.046 1/м |
| Нп/м | μ_Np = μ_dB/4.343 | 0.046 Нп/м |
| FDS OD [1/м] | μ_FDS = μ_dB/10 | 0.02 1/м |
| FDS %/м | OD_% = (1-e^{-μ_dB/4.343})·100 | 4.5 %/м |

#### Из FDS OD [1/м]

| В какую величину | Формула | Пример: μ_FDS=0.02 |
|------------------|---------|--------------------|
| 1/м (K) | K = 2.3·μ_FDS | 0.046 1/м |
| Нп/м | μ_Np = 2.3·μ_FDS | 0.046 Нп/м |
| дБ/м | μ_dB = 10·μ_FDS | 0.2 дБ/м |
| FDS %/м | OD_% = (1-10^{-μ_FDS})·100 | 4.5 %/м |

#### Из FDS %/м

| В какую величину | Формула | Пример: OD_%=4.5 |
|------------------|---------|------------------|
| 1/м (K) | K = -ln(1-OD_%/100) | 0.046 1/м |
| Нп/м | μ_Np = -ln(1-OD_%/100) | 0.046 Нп/м |
| дБ/м | μ_dB = -10·log₁₀(1-OD_%/100) | 0.2 дБ/м |
| FDS OD [1/м] | μ_FDS = -log₁₀(1-OD_%/100) | 0.02 1/м |

## 2. Практические примеры для Методики 1140

### Пороговые значения из нормативных документов

| Источник | Значение | Ед. изм. | В FDS %/м | В FDS OD [1/м] | В K [1/м] |
|----------|----------|----------|-----------|----------------|-----------|
| Методика 1140 (дым) | 0.023 | Нп/м | 2.27 | 0.01 | 0.023 |
| Методика 1140 (дым) | 0.2 | дБ/м | 4.50 | 0.02 | 0.046 |
| FDS по умолчанию | 3.24 | %/м | 3.24 | 0.014 | 0.033 |
| EN 54-7 | 0.5 | дБ/м | 10.9 | 0.05 | 0.115 |
| NFPA 72 | 4.0 | %/м | 4.0 | 0.018 | 0.041 |

### Примеры расчёта

**Пример 1:** Порог 0.023 Нп/м (Методика 1140, дымовые извещатели)

```
Дано: μ_Np = 0.023 Нп/м

Решение:
K = 0.023 1/м
μ_dB = 4.343 × 0.023 = 0.10 дБ/м
μ_FDS = 0.023 / 2.3 = 0.01 1/м
OD_% = (1 - e^(-0.023)) × 100 = 2.27 %/м

Ответ: В FDS задать ACTIVATION_OBSCURATION = 2.3 %/м
```

**Пример 2:** Порог 0.2 дБ/м (Методика 1140, альтернативное значение)

```
Дано: μ_dB = 0.2 дБ/м

Решение:
K = 0.2 / 4.343 = 0.046 1/м
μ_Np = 0.046 Нп/м
μ_FDS = 0.2 / 10 = 0.02 1/м
OD_% = (1 - e^(-0.046)) × 100 = 4.5 %/м

Ответ: В FDS задать ACTIVATION_OBSCURATION = 4.5 %/м
```

**Пример 3:** FDS по умолчанию 3.24 %/м

```
Дано: OD_% = 3.24 %/м

Решение:
K = -ln(1 - 3.24/100) = 0.033 1/м
μ_Np = 0.033 Нп/м
μ_dB = 4.343 × 0.033 = 0.143 дБ/м
μ_FDS = -log₁₀(1 - 3.24/100) = 0.014 1/м

Ответ: Это соответствует 0.143 дБ/м или 0.033 Нп/м
```

## 3. Быстрый калькулятор

### Python-функции для преобразования

```python
import numpy as np

def to_FDS_from_Np(mu_Np):
    """Из Нп/м в FDS %/м"""
    return (1 - np.exp(-mu_Np)) * 100

def to_FDS_from_dB(mu_dB):
    """Из дБ/м в FDS %/м"""
    K = mu_dB / 4.343
    return (1 - np.exp(-K)) * 100

def from_FDS_to_Np(OD_percent):
    """Из FDS %/м в Нп/м"""
    return -np.log(1 - OD_percent/100)

def from_FDS_to_dB(OD_percent):
    """Из FDS %/м в дБ/м"""
    return -10 * np.log10(1 - OD_percent/100)

def to_FDS_OD_from_Np(mu_Np):
    """Из Нп/м в FDS OPTICAL DENSITY [1/м]"""
    return mu_Np / 2.3

def to_FDS_OD_from_dB(mu_dB):
    """Из дБ/м в FDS OPTICAL DENSITY [1/м]"""
    return mu_dB / 10

def from_FDS_OD_to_Np(mu_FDS):
    """Из FDS OPTICAL DENSITY [1/м] в Нп/м"""
    return mu_FDS * 2.3

def from_FDS_OD_to_dB(mu_FDS):
    """Из FDS OPTICAL DENSITY [1/м] в дБ/м"""
    return mu_FDS * 10

# Примеры использования
print(f"0.023 Нп/м = {to_FDS_from_Np(0.023):.2f} %/м (FDS)")
print(f"0.2 дБ/м = {to_FDS_from_dB(0.2):.2f} %/м (FDS)")
print(f"3.24 %/м (FDS) = {from_FDS_to_Np(3.24):.3f} Нп/м")
print(f"3.24 %/м (FDS) = {from_FDS_to_dB(3.24):.3f} дБ/м")
```

### Excel-формулы

| Преобразование | Формула Excel |
|----------------|---------------|
| Нп/м → %/м | `=(1-EXP(-A1))*100` |
| дБ/м → %/м | `=(1-EXP(-A1/4.343))*100` |
| %/м → Нп/м | `=-LN(1-A1/100)` |
| %/м → дБ/м | `=-10*LOG10(1-A1/100)` |
| Нп/м → FDS OD [1/м] | `=A1/2.3` |
| дБ/м → FDS OD [1/м] | `=A1/10` |
| FDS OD [1/м] → Нп/м | `=A1*2.3` |
| FDS OD [1/м] → дБ/м | `=A1*10` |

## 4. Массовый коэффициент экстинкции

### Типовые значения

| Материал | K_m [м²/кг] | Источник |
|----------|-------------|----------|
| Сажа (по умолчанию FDS) | 8700 | FDS |
| Древесина (тление) | 345 | ГОСТ 12.1.044 |
| Древесина (горение) | 23 | ГОСТ 12.1.044 |
| Полиэтилен | 890 | ГОСТ 12.1.044 |
| Полистирол | 1048 | ГОСТ 12.1.044 |
| ПВХ | 400-1290 | ГОСТ 12.1.044 |
| Резина | 850 | ГОСТ 12.1.044 |

### Расчёт коэффициента экстинкции

$$K = Y_{soot} \cdot \rho \cdot K_m$$

где:
- $Y_{soot}$ — массовая доля сажи [безразмерная]
- $\rho$ — плотность газовой смеси [кг/м³]
- $K_m$ — массовый коэффициент экстинкции [м²/кг]

**Пример:**
```
Y_soot = 0.001 (0.1% сажи)
rho = 1.2 кг/м³
K_m = 8700 м²/кг

K = 0.001 × 1.2 × 8700 = 10.44 1/м

В FDS:
EXTINCTION COEFFICIENT = 10.44 1/м
OPTICAL DENSITY = 10.44/2.3 = 4.54 1/м
```

## 6. Шпаргалка для настройки FDS

### Для дымовых извещателей

```fortran
&PROP ID='Smoke_Detector', 
      ACTIVATION_OBSCURATION=2.3 /  ! 0.023 Нп/м (Методика 1140)
```

или

```fortran
&PROP ID='Smoke_Detector', 
      ACTIVATION_OBSCURATION=4.5 /  ! 0.2 дБ/м (Методика 1140)
```

### Для тепловых извещателей

```fortran
&PROP ID='Heat_Detector', 
      ACTIVATION_TEMPERATURE=343.15 /  ! 70°C
```

### Для измерительных устройств

```fortran
! Коэффициент экстинкции [1/м]
&DEVC ID='EXT-1', 
      QUANTITY='EXTINCTION COEFFICIENT', 
      SPEC_ID='SOOT',
      XYZ=2.5,2.5,2.9 /

! Оптическая плотность [1/м] (десятичные логарифмы)
&DEVC ID='OD-1', 
      QUANTITY='OPTICAL DENSITY', 
      SPEC_ID='SOOT',
      XYZ=2.5,2.5,2.9 /

! Температура [K]
&DEVC ID='TEMP-1', 
      QUANTITY='TEMPERATURE', 
      XYZ=2.5,2.5,2.9 /
```
